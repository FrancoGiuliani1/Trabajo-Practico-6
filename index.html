      • ¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.
    • Manejo de variables:
      • Hay tres tipos de variables estas son var, let, const.
      • var: Nos permite crear variables que pueden ser modificadas.
      • let: Esta variables nos permiten modificarlas, pero no se pueden re-declarar. Tambien puede servirnos para escribir un codigo mas limpio al usar funcines internas.
      • const: Se utiliza para crear variables de solo lectura, quiere decir que una vez que se crea esta no puede ser modificada.
    • Tipos de datos:
      • Undefined:Es cuando a una variable no se le asigna un valor como tal, estas son de tipo Undefined.
      • Boolean: Este puede presentar dos valores como tal. Es decir, comprueban que una condicion sea "esto o esto". Por ejemplo: verdaro o falso
      • Number: Este tipoo de dato es numerico y consta de 64 bits. Tiene tres valores: +Infinity, -Infinity y NaN ("Not a Number" o No es un número).
      • String: Se utiliza para crear cadenas, el primer elemento tiene la posición 0 y el segundo la posición 1.
      • Bigint: Se utiliza para almacenar y operar grandes numeros de una forma segura.
      • Symbol: Este es un valor unico y no se puede cambiar o sobreescribir y se puede utilizar como la clave una propiedad de objeto.
      • Object:Se utilizan para almacenar varios datos y colecciones de entidades más complejas asociadas con nombres clave. En JavaScript, los objetos impregnan casi todos los aspectos del lenguaje.
      ¿Cómo son todas las formas de declaración de funciones en JS?.
    • Las diferentes formas de declarar funciones son:
      • Function declaration: Este tipo de función se creará utilizando la palabra function, seguida de un nombre que identifique nuestra función, una lista de variables dentro de paréntesis y una llave {}. Las cuales determinaran el fin de la funcion.
      • Function expression: La sintaxis para crear estas funciones es similar a Function declaration. La diferencia entre estas es su definición de nuestra función no comienza por instrucción function y el nombre de la función es opcional. Este tipo de funciones las podemos almacenar en una variable.
      • IIFE (Immediately Invoked Function Expression): Este tipo de funciones se ejecutan inmediatamente y no son accesibles posteriormente. Es decir, solo se pueden utilizar una sola vez.
      • Shorthand method definition:
      • Arrow function: Esta función puede ser utilizada como método en la declaración de un objeto o en las clases de ES6.
      • Generator function: Esta funcion nos permite acortar la funcion a una sola linea y que el codigo sea mucho mas "limpio", ya que dejamos de lado el return.
      • Function constructor: Esta es una función especial que crea e inicializa una instancia de objeto de una clase. En JavaScript, se llama a un constructor cuando se crea un objeto usando la nueva palabra clave 'this'. El propósito es crear un nuevo objeto y establecer valores para cualquier propiedad de objeto existente.
